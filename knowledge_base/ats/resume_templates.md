# Resume Templates

This document describes common resume structures, when to use them, and their advantages.

---

# Reverse Chronological Resume (Recommended)

The reverse chronological resume is the most widely accepted resume format and is preferred by most recruiters and Applicant Tracking Systems (ATS).

Structure:

1. Contact Information
2. Professional Summary or Objective
3. Work Experience (Most Recent First)
4. Projects
5. Education
6. Skills
7. Certifications
8. Additional Sections (Achievements, Volunteer Work, Languages)

Best for:
- Students
- Fresh graduates
- Professionals with consistent work experience

Advantages:
- ATS-friendly
- Easy for recruiters to scan
- Clearly shows career progression
- Highlights recent experience first

---

# Functional Resume

A functional resume emphasizes skills instead of employment history.

Structure:

1. Contact Information
2. Professional Summary
3. Core Skills
4. Projects / Achievements
5. Work History
6. Education

Best for:
- Career changers
- Large employment gaps
- Freelancers

Advantages:
- Highlights transferable skills
- Reduces focus on work gaps

Disadvantages:
- Less ATS-friendly
- Recruiters often prefer chronological resumes

---

# Combination Resume

A combination resume blends skill-focused sections with chronological work history.

Structure:

1. Contact Information
2. Professional Summary
3. Technical Skills
4. Work Experience
5. Projects
6. Education
7. Certifications

Best for:
- Experienced professionals
- Technical roles
- Candidates with diverse experience

Advantages:
- Shows both technical skills and work history
- Suitable for experienced candidates

---

# Recommended Resume Structure for Software Engineers

1. Contact Information
2. Professional Summary
3. Technical Skills
4. Projects
5. Experience
6. Education
7. Certifications
8. Achievements
9. Links (GitHub, Portfolio, LinkedIn)

Projects should include:
- Technologies used
- Problem solved
- Personal contribution
- Measurable impact when possible

---

# Resume Length Guidelines

Students:
- One page

Junior professionals:
- One page (maximum two)

Experienced professionals:
- Two pages

Avoid unnecessary information simply to increase resume length.

---

# Section Order Recommendation

For students:

1. Contact Information
2. Objective / Summary
3. Education
4. Projects
5. Skills
6. Certifications
7. Leadership / Achievements

For experienced professionals:

1. Contact Information
2. Professional Summary
3. Experience
4. Projects
5. Skills
6. Education
7. Certifications

# ATS-Friendly Template Checklist

✓ Single-column layout

✓ Standard section headings

✓ Reverse chronological ordering

✓ Standard fonts

✓ Consistent date formatting

✓ Bullet points

✓ Text-based PDF

✓ No graphics or decorative elements
